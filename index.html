Configurar Google Sheets

Para conectar o app à sua planilha, você precisa de um ID da planilha e de uma chave de API do Google.

Como obter em ~10 minutos:
  1. Abra sua planilha no Google Sheets
  2. Copie o ID da URL: docs.google.com/spreadsheets/d/[ID AQUI]/edit
  3. Acesse console.cloud.google.com → crie projeto → ative Google Sheets API
  4. Crie uma credencial do tipo Chave de API e restrinja ao Sheets API
  5. Cole os dois campos abaixo e clique em Salvar
